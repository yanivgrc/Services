# Screensaver — GET command scripts (use verbatim)

Each command in the screensaver terminal should render as a **winget-style
install** followed by the **full response** below. This is stylized demo content
— a showcase of the service, not a real installer. Keep the `// demo` spirit.

## Prompt label
Use `grc-labs:\>` as the prompt (not `KPEUS:\>`).

## winget-style install preamble (before each response)
Render something like this, typed out, before the full text of each command:

```
grc-labs:\> get <NAME>
Found <NAME> [GRC-Labs.<NAME>]
Downloading <module-name> ...
██████████████████████████  100%
Successfully installed. Launching ...
```

Then print the matching full response below. Cycle a DIFFERENT command each time.
